# 🌾 Smart Agriculture Dashboard

## 📌 Overview
An interactive dashboard built using R Shiny for agriculture data analysis.

## 🚀 Features
- Crop production analysis  
- Rainfall visualization  
- State-wise insights  
- Interactive charts  

## 🛠 Tech Stack
- R  
- Shiny  
- ggplot2
